# GPS 軌跡紀錄器 GitHub Pages 版

這份版本已依照你的 GitHub Repository 設定好：

- GitHub 帳號 / 組織：`fox520-sketch`
- Repository：`qps_track`
- 預計 GitHub Pages 網址：`https://fox520-sketch.github.io/qps_track/`

## 部署方式

1. 建立或使用 GitHub repo：`fox520-sketch/qps_track`
2. 把這個壓縮包解開後的所有檔案，上傳到 repository 根目錄
3. 推送到 `main` 分支
4. 到 GitHub 的 `Settings` → `Pages`
5. 啟用 GitHub Pages
   - 可使用內附的 GitHub Actions workflow 自動部署
   - 或直接設定由 `main` 分支 root 發佈
6. 發佈完成後，開啟：
   `https://fox520-sketch.github.io/qps_track/`

## 手機安裝

- Android Chrome：選單 → 加到主畫面
- iPhone Safari：分享 → 加入主畫面

## 注意

- 建議用 GitHub Pages 的 HTTPS 網址開啟
- 第一次載入地圖仍需要網路
- 手機背景執行時，瀏覽器仍可能限制定位更新頻率
- 之後若改用自訂網域，可再加入 `CNAME`
